1. Executar o servidor:
python -m server.server

2. Executar o cliente:
python -m client.client

3. Introduzir o número do bloco pretendido

4. O cliente recebe o bloco serializado em formato JSON

5. O cliente guarda o bloco num ficheiro

6. O cliente pode reenviar o bloco ao servidor para desserialização

7. O servidor processa e devolve a informação do bloco
